Important classes¶

class camera¶

Represents the camera in a 2.5D raycasting engine.

Public Functions

camera(sf::RenderWindow &renderWindow, map &Map)¶

Constructs a camera object.

Parameters:

  • renderWindow – Reference to the SFML window where the scene is rendered.

  • Map – Reference to the map object used by the camera.

void draw3D()¶

Renders the 3D view of the scene based on the camera’s perspective.

void update()¶

Updates the camera’s position and orientation based on user input.

void projection()¶

Casts rays from the camera’s position to determine distances to obstacles.

void checkKeyboardHit(sf::Time dt)¶

Handles keyboard input to control the camera’s movement and rotation.

Parameters:

dt – Delta time to ensure consistent movement speed regardless of frame rate.

bool intersect(unsigned int it)¶

Determines whether a ray intersects with an obstacle in the map.

Parameters:

it – The index of the ray.

Returns:

true if the ray intersects with an obstacle, false otherwise.

inline float dSin(float deg)¶

Computes the sine of an angle in degrees.

class map¶

Represents the game map, including its dimensions and obstacles.

The map class handles the creation of a game environment, including generating obstacles and providing methods to interact with the map data.

Public Functions

map(sf::RenderWindow &m_renderWindow)¶

Constructs a map object.

Initializes the map, sets its dimensions, and populates it with obstacles.

Parameters:

m_renderWindow – Reference to the SFML render window.

bool checkMapCase(unsigned int dx, unsigned int dy)¶

Checks if a specific map cell is a wall or obstacle.

Parameters:

  • dx – The x-coordinate of the cell.

  • dy – The y-coordinate of the cell.

Returns:

true if the cell is a wall or obstacle, false otherwise.

size_t getWidth()¶

Gets the width of the map.

Returns:

The width of the map in blocks.

size_t getHeight()¶

Gets the height of the map.

Returns:

The height of the map in blocks.
